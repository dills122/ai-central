# Security

This repo contains AI guidance, scaffolding scripts, and copied project context. Treat it as potentially sensitive.

## Sensitive Content Review

Before publishing or sharing this repository:

- Review `collected/` for private project details.
- Remove secrets, tokens, internal hostnames, customer data, and proprietary implementation details.
- Check temporary third-party imports for licensing and provenance.
- Rebuild `docs/source-manifest.sha256` after any collection changes.

## Reporting

For local personal use, file issues directly in this repository or fix the content in place.

For anything that may expose credentials or private infrastructure, rotate the affected credential first, then update the collected material.
